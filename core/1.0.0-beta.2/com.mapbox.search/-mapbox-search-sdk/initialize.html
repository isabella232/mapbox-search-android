initialize

@JvmOverloads @JvmStatic fun initialize(application: Application, accessToken: String, locationProvider: LocationProvider, geocodingEndpointBaseUrl: String = DEFAULT_ENDPOINT_GEOCODING, singleBoxSearchBaseUrl: String = DEFAULT_ENDPOINT_SBS, viewportProvider: ViewportProvider? = null): Unit

Experimental API, can be changed or removed in the next SDK releases. Initialize Search SDK.

Parameters

application - android application instance.

accessToken - mapbox access token.

locationProvider - location provider instance.

geocodingEndpointBaseUrl - Geocoding API endpoint URL. This parameter is optional, default URL is https://api.mapbox.com.

singleBoxSearchBaseUrl - Single Box Search endpoint URL. This parameter is optional, default URL is https://sbs-beta.mapbox.com.

viewportProvider - optional viewport provider.

Exceptions

IllegalStateException - if MapboxSearchSdk has already been initialized.
